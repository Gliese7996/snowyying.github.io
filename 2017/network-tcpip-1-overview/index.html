地址 与 域名

网络号 + 主机号 (32 Bit)

  • 单播地址: 目的端为单个主机
  • 组播地址: 目的端为同一组内所有主机

  • 广播地址: 目的端为给定网络上所有主机

  • A类: 0+7bit(网络号)+24bit(主机号); 0.0.0.0 ~ 127.255.255.255

  • B类: 10+14bit(网络号)+16bit(主机号); 128.0.0.0 ~ 191.255.255.255
  • C类: 110+21bit(网络号)+8bit(主机号); 192.0.0.0 ~ 223.255.255.255
  • D类: 1110+28bit(多播组号); 224.0.0.0 ~ 239.255.255.255
  • E类: 11110(Reserved); 240.0.0.0 ~ 255.255.255.255
