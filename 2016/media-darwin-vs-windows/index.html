Visual Studio 编译 Darwin Streaming Server

2016-12-20
Media

1 Patch

2 Fix compile error

1) OSHeaders.h
  • 解决socket相关函数重复定问题;
  • 解决time相关函数编译失败的问题.
2) 宏定义

搜索解决方案, 在_S32BITARG_ / _U32BITARG_ / _64BITARG_ / _S64BITARG_ / _U64BITARG_前后加空格.

在VS2015下宏定义必须前后带空格符, 否则会被识别成字符串导致编译失败.

3) arpa/inet.h

搜索解决方案, 对#include 进行修改, 在Windows上平台上没有此头文件.

4) QTAccessFile.cpp

注释掉以下头文件的引用, 同时, 注释掉一个未被引用的函数.

5) QTSSPasswd.cpp

打开此注释

6) RTPStream.cpp

3 Fix Link error

链接错误是因为找不到对应的实现, 给链接不通过的工程添加实现即可, 不再赘述.

  • StreamingLoadTool (需添加1个cpp文件到工程)
  • StreamingServer (需添加N个cpp文件到工程)
